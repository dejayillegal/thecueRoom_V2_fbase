This directory stores Expo assets for the mobile app.

Fonts
-----
Inter and Source Code Pro are loaded via `@expo-google-fonts`. If you need offline usage,
place downloaded `.ttf` files here and update `App.tsx` to load them with `Font.loadAsync`.

SVGs
----
Place SVG artwork here and import it directly (see `docs/SVG_TRANSFORMER.md` for the transformer pipeline).
Keep filenames descriptive, e.g. `MarketingLanding.svg`.
